An Image-Based Multifunctional Fruit Classification System Using MobileNet CNN for Tensorflow

This study is entitled Automated Fruit Classification System with Quality Grading and Sizing Using Digital Image Processing, this study is a multifunctional fruit conveyor system with sorting mechanisms with raspberry pi microprocessor as the brain of the system and a raspberry pi camera module v2 as camera to identify the name of the fruit, detect the quality of the fruit, e.g. good quality, damage, and unripe and sort the size of the fruit, e.g. small, medium, and large. Mobilenet CNN is a light-weight machine learning algorithm design to perform convolutional neural network processes for low processing speed devices such as raspberry pi microprocessor, this machine learning algorithm is used to train a fruit classification model that can detect, identify, and grade the quality of the fruits, using image processing algorithm as a means to detect the size of the fruit, the number of pixels the image consume is identified and use to detect the size of fruit being processed.
